# nvim config 

```sh
# on Linux and Mac
git clone https://github.com/ThomasFerro/dot.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```


```
# on Windows (cmd)
git clone https://github.com/ThomasFerro/dot.git %userprofile%\AppData\Local\nvim\ 
```

```
# on Windows (powershell)
git clone https://github.com/ThomasFerro/dot.git $env:USERPROFILE\AppData\Local\nvim\ 
```
